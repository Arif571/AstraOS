# ⭐ AstraOS

> *"As high as stars, illuminating all users"*

AstraOS is a next-generation Linux distribution built for everyone —
developers, security researchers, desktop users, and data scientists.

## ✨ Key Features
- 🧑‍💻 **DevMode** — Full development environment
- 🛡️ **ShieldMode** — Security & privacy focused  
- 🖥️ **DesktopMode** — Beautiful & easy to use
- 🔬 **LabMode** — Data science & ML ready
- ⚡ One command to switch between modes: `astra switch --mode <name>`

## 🏗️ Built on
- Base: Debian Stable
- DE: KDE Plasma / i3
- Kernel: Hardened Linux
